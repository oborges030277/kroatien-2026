## Project Overview

A vacation planning website for a Croatia trip (July 18 – August 1, 2026) to Villa Lavanda Macarini in Istria. The site is a single-page German-language HTML page with embedded CSS/JS, generated by a Python build script.

## Build

```bash
python build_final.py
```

This reads image data from `extracted_images/compressed.json` and `extracted_images/web_images.json`, then generates `index.html` (a self-contained ~4.5MB HTML file with base64-embedded images).

The script uses relative paths based on its own location, so it works from any directory.

## Architecture

- **`build_final.py`** — The authoritative build script. Generates the complete `index.html` with all CSS, JS, and image data inlined. Contains three main parts:
  1. Image loading from JSON files (base64-encoded images from the PDF + web-sourced images)
  2. CSS and JS as Python string constants (`CSS`, `JS`)
  3. HTML template as an f-string, using the `img()` helper to insert base64 data URIs
- **`index.html`** — Generated output. Do not edit directly; modify `build_final.py` and rebuild.
- **`build_html.py` / `build_html_v2.py`** — Earlier iterations of the build script. Not used; `build_final.py` is the current version.
- **`extracted_images/`** — Images extracted from the PDF, plus JSON manifests (`compressed.json`, `web_images.json`) mapping image keys to base64 strings.
- **`villa_fotos/`, `villa_fotos_fewo/`, `beach_fotos/`** — Source photo directories.
- **`Urlaub 2026 - Kroatien.pdf`** — Original vacation planning PDF.

## Deployment

The site is hosted on GitHub Pages: https://oborges030277.github.io/kroatien-2026/

After rebuilding `index.html`, commit and push to `main` to deploy.

## Key Details

- The page is entirely in German.
- All images are base64-embedded — there are no external image references.
- CSS uses short class names (e.g., `.st` for section title, `.cb` for card body, `.cg` for card grid) defined in the `CSS` constant.
- The JS includes a countdown timer targeting `2026-07-18T15:00:00+02:00` and a lightbox for image zoom.
- The site sections cover: villa details, beaches, dog-friendly beaches, restaurants, weather, route/rest stops, attractions, a map, and travel tips.
